# Установка раздела "Отчеты"

## 🚀 Быстрая установка

### Шаг 1: Установите зависимости
```bash
npm install
```

Эта команда установит новую библиотеку `recharts` для графиков.

### Шаг 2: Запустите приложение
```bash
npm run dev
```

### Шаг 3: Откройте в браузере
```
http://localhost:3000
```

## ✅ Проверка работоспособности

1. Откройте приложение
2. В Header должны быть две вкладки: **Склад** и **Отчеты**
3. Кликните на **Отчеты**
4. Должна открыться страница с фильтрами времени и 4 вкладками
5. Выберите период (например, "Последние 24 часа")
6. Нажмите **"Обновить"**
7. Должны загрузиться данные и отобразиться графики

## 🔧 Если возникли проблемы

### Ошибка: "recharts is not defined"
```bash
npm install recharts
```

### Ошибка: "Cannot find module '@/types/reports'"
Проверьте, что файл `src/types/reports.ts` создан.

### Ошибка сети при загрузке отчетов
Проверьте:
1. Доступен ли API: `http://10.171.12.36:3006`
2. Работает ли endpoint: `GET /api/logs/report`
3. Есть ли данные в таблице `x_API_Operation_Logs`

### Пустые графики
Это нормально, если в базе данных нет логов операций. 
Выполните несколько операций на складе, и они появятся в отчетах.
