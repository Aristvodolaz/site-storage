# 🚀 Быстрый старт: Раздел "Отчеты"

## Что нужно сделать

### 1. Установите библиотеку recharts
```bash
npm install
```

### 2. Запустите приложение
```bash
npm run dev
```

### 3. Откройте и используйте
1. Откройте http://localhost:3000
2. Кликните на вкладку **"Отчеты"** в Header
3. Выберите период времени
4. Нажмите **"Обновить"**
5. Смотрите отчеты в 4 вкладках:
   - **Активность по времени** - графики операций
   - **Операции (ЧТО)** - что делали
   - **Пользователи (КТО)** - кто делал
   - **Детали (КТО+ЧТО+ГДЕ)** - полная информация

## Готово!

Теперь вы можете видеть:
- ✅ **КТО** делал операции (пользователи)
- ✅ **ЧТО** делали (операции/endpoints)
- ✅ **ГДЕ** делали (endpoints с полными путями)
- ✅ **КОГДА** делали (фильтры по времени)
- ✅ Статистику успешности и времени выполнения
