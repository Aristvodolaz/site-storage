# Реализация раздела "Отчеты"

## 📋 Что было добавлено

Создан полноценный раздел отчетов для просмотра операций с информацией:
- **КТО** - какой пользователь выполнил операцию
- **ЧТО** - какая операция была выполнена
- **ГДЕ** - на каком endpoint

## 🎯 Функциональность

### 1. Навигация
- Добавлены вкладки в Header: **Склад** и **Отчеты**
- Переключение между разделами через клик на вкладки

### 2. Фильтры по времени
- **Последние 24 часа** - быстрый просмотр текущей активности
- **Последние 7 дней** - недельный отчет
- **Последние 30 дней** - месячный отчет
- **Сегодня** - операции за текущий день
- **Вчера** - операции за вчерашний день
- **Произвольный период** - выбор любого диапазона дат

### 3. Четыре вкладки с отчетами

#### 📊 Активность по времени
- График операций по часам/дням
- Визуализация успешных операций и ошибок
- График Success Rate и среднего времени выполнения

#### 📝 Операции (ЧТО)
- Таблица топ-20 самых частых операций
- Показывает endpoint каждой операции
- Статистика по каждой операции: количество, успешность, время выполнения
- Количество уникальных пользователей на операцию

#### 👥 Пользователи (КТО)
- Pie chart распределения операций по пользователям
- Детальная таблица со статистикой по каждому пользователю
- Success rate для каждого пользователя
- Количество IP адресов

#### 🔍 Детали (КТО+ЧТО+ГДЕ)
- Полная таблица с детальными логами
- Информация о каждой операции: время, пользователь, метод, endpoint, статус
- Цветовая индикация статусов и методов
- Время выполнения каждой операции

## 🚀 Установка и запуск

### 1. Установите зависимости
```bash
npm install
```

Это установит новую библиотеку `recharts` для графиков.

### 2. Запустите приложение
```bash
npm run dev
```

### 3. Откройте в браузере
```
http://localhost:3000
```

## 📊 Использование

1. Откройте приложение
2. В Header кликните на вкладку **"Отчеты"**
3. Выберите период времени (по умолчанию: последние 24 часа)
4. Нажмите **"Обновить"** для загрузки данных
5. Переключайтесь между вкладками для просмотра разных отчетов

## 🎨 Визуализация

### Карточки сводки
- Всего операций
- Успешных операций (с процентом успеха)
- Ошибок (с процентом ошибок)
- Среднее время выполнения (с максимальным временем)

### Графики
- **Bar Chart** - количество успешных операций и ошибок по времени
- **Line Chart** - Success Rate и среднее время выполнения
- **Pie Chart** - распределение операций по пользователям

### Таблицы
- Цветовая индикация Success Rate (зеленый >95%, желтый >80%, красный <80%)
- Цветовая индикация HTTP статусов
- Цветовая индикация HTTP методов
- Сортировка по количеству операций

## 🔧 Технические детали

### API Integration
Приложение использует endpoint:
```
GET http://10.171.12.36:3006/api/logs/report
```

### Параметры запроса
- `date_from` - начало периода (ISO 8601)
- `date_to` - конец периода (ISO 8601)
- `group_by` - тип группировки (hour, day, endpoint, executor, status_code, method)
- `include_details` - включить детальные логи (опционально)

### Кэширование
- Данные кэшируются на 2 минуты (React Query)
- Автоматическое обновление при изменении фильтров
- Ручное обновление кнопкой "Обновить"
